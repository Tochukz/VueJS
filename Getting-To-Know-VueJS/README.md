__Vue Basic commands__   
* Install the Vue CLI  
`$ npm install -g @vue/cli`    
* Create Vue Projects    
`$ vue create my-app`   
* Serve the project using Vue-CLI-Service   
`$ npm run serve`  
* Run Build for production    
`$ npm run build`      
* Lunch Vue GUI  
`$ vue ui`
